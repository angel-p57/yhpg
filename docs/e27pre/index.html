Light and mirrors #yhpg e27pre 2018.10.6

Abstract

Given that there exist 5x5 squares named "a" thru "y" like the figure at right.
Each square is one of;

  • a blank
  • one with a double sided mirror placed along the diagnol line whose left end is up and right end is down
  • one with a double sided mirror placed along the diagnol line whose right end is up and left end is down
  • one absorbing any light
  • the place you stand

When you beam a light up straight ( to the north direction ), find the all squares at which the light pass through.

Mission

Your mission is to write a program which solves this problem.

You can write one in any language, and you can execute it on any environment ( i.e, on your PC, on some online IDE, etc. ), and you can spend as much CPU time or memory as you can prepare. There is no restrictions.

And you can choose, as you like, how the program receives the input parameters and outputs the answers.
For example, angel_p_57's Brainf**k program receives input parameters from STDIN, and simply output the answers to STDOUT.
But I prepared a test template for C/C++/Java and a JSON test data file both of which contain all input parameters and the correct answers. They could be helpful for you.

After you solve this problem, I'd be very glad if you'd make a comment, including a link to your source, at the article in Qiita.

Input parameter's format

Each input parameter, which describes a set of 5x5 squares, is a string like this;
x...x/.1.0./..0../.Y.../0..x.
The string is slash-separated, and each separated chunk, which describes the corresponding row in the squares, consists of 5 ASCII characters mapped to the types of the squares in the row.
Each chunks appeared in a top-to-down ( north-to-south ) order, and each characters in a left-to-right ( west-to-east ) order.
The mapping is as follows;

Character Square type
. A blank. This absolutely pass through light.
0 One with a double sided mirror placed along the diagnol line whose left end is up and right end is down
1 One with a double sided mirror placed along the diagnol line whose right end is up and left end is down
x One absorbing any light
Y The place you stand. This also pass through light.

Output format

Each output ( answer ) should be a string like this;
ghilnqs
It should have all the alphabet names of the squares at which the light pass through, with no duplicates, in a alphabetical ascending order.

Additional Info

  • There is no invalid input parameters. You can count on it.
  • You are alone in each case. In other words, there is one and only one square at which you stands.
  • Don't think if the light may fade or diffuse.
  • All the angles between the mirrors and the edges of the squares are exactly 45 degree.
  • If the light go out of 5x5 squares, then it won't come back. You can treat it disappeared.
  • This problem is somewhat a trial for the next #yhpg e27. The problem will be easy/difficult as this problem.
  • #yhpg e27 is scheduled on 2018.10.6 in Tokyo. Refer to the event information page in Doorkeeper.
